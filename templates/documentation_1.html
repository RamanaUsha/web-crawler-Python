Project Documentation

Setup Instructions

Follow these steps to set up the project:

git clone 
cd 
pip install -r requirements.txt

Project Structure

File Description
app.py Main application file
templates/ HTML templates for web pages
static/css/ CSS stylesheets
data.csv Storage for crawl history and results

Usage Guide

  1. Navigate to the homepage and log in or sign up.
  2. Enter a URL, set crawl depth, max pages, and specify keywords (optional).
  3. Click "Start Crawl" to initiate. Results appear on the results page.
  4. Check the crawl history to view previous crawls.

API Endpoints

  • /crawl: Initiates the crawling process
  • /history: Displays user crawl history
  • /profile: Shows user profile information
